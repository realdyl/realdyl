## 📊 JEXXXUS | Ecosystem Repos

| Repo | Purpose | Stack |
|------|---------|-------|
| **blxckbook.jexxx.us** | Core platform engine | React 18 + Vite + Tailwind |
| **dxsh.blxckbook.jexxx.us** | Admin control plane | React 19 + Radix UI + GSAP |
| **nxt.jexxx.us** | AI persona engine | React + Gemini + ChromaDB |
| **blxckbook-mobile** | Pocket app | React Native + Expo + Nativewind |
| **jexxx.us-cli** | Headless executor | TypeScript + Commander |

All repos share authentication (Clerk), data (Supabase), and types via `@blxckbook/shared-types` monorepo packages.
